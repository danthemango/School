# checkerboxes
A python3 stript which prints all possible combinations of 2xN chessboards covered only by square and corner tiles
```
             ┌┐               ┌┐
square tile: └┘ corner tile:  │└┐
                              └─┘
```

## Usage

% python printBox.py -h
usage: printBox.py [-h] -n N [-c N]

MATH223 Ass.4: print the checker boards of size 2xN (given by -n), covered
only by square and corner tiles

optional arguments:
  -h, --help      show this help message and exit
  -n N, --num N   print boxes of size 2xN
  -c N, --cols N  number of boxes to print before a linebreak
